Grid Framework  2.1.6
Easy grid setup and scripting for Unity 3D
Overview

Preface

Thank you for choosing Grid Framework for Unity3D. My goal was to create an easy to use setup where the users only needs to say what kind of grid they want and let the computer do all the math. After all, that is why computers where invented in the first place.

This package provides you with new Grid components which can be added to any GameObject in the scene, Renderer components for the grids, lots of functions to use with the grids and an editor panel for aligning and scaling objects in your scene. Every grid is three-dimensional and infinite in size. You can draw your grids in the editor using gizmos, render them at runtime and even use Grid Framework along with Vectrosity and Playmaker if you have a license for it.

By buying this package you support further development and you are entitled to all future upgrades for free.

Alejandro "HiPhish" Sanchez

Setup

Nothing special is needed if you get Grid Framework form the Unity Asset Store. If you accidentally moved some files and are getting compilation errors you can restore things back to normal by placing the files in the same locations as described below.

All features of Grid Framework, including the documentation, can be accessed through the menu bar, see the Getting Started page for more information, there is no reason to attach scripts manually.

What is included

The package contains one main directory Plugins/Grid Framework with further sub-directories. Out of these the Examples directory is the one most relevant to end users, it contains a number of example scenes showcasing Grid Framework in practice. The main directory also contains the same changelog as this document.

You can safely delete the examples, but you don't have to; all examples are namespaced and will not interfere with your own code. Every example comes with a readme file that explains the idea behind the example and how it has been implemented (the readme can be read using any plain text editor).

The Vectrosity examples need a copy of Vectrosity installed and Vectrosity support enabled in order to work. Vectrosity is a separate vector line drawing extension for Unity written by Eric5h5 and not related to Grid Framework. Please see the rendering section for more information.

The Documentation directory contains this document in various file formats. If you delete the sub-directory the Help menu item will take you to an online version of it.

The remaining directories are source files of Grid Framework and are required for working. Please do not modify them directly or your changes will be overwritten when updating. It is better to write your own grids, renderers and extension methods instead, that way your custom code remains compatible with all future updates.
