Enumerator
World 

World-space coordinates.

Grid 

A rectangular grid that uses Side and Height as its width and height.

This coordinate system does not look hexagonal and is only used as an intermediate step.

HerringboneUp 

Herringbone pattern where every odd column is shifted upwards.

HerringboneDown 

Herringbone pattern where every odd column is shifted downwards.

RhombicUp 

Rhombic pattern where both axes go along the flat sides at a 60° angle towards each other.

RhombicDown 

Rhombic pattern where both axes go along the flat sides at a 300° angle towards each other.

Cubic 

Cubic pattern with three axes in a plane, each one going along the flat sides, at 60° to each other. The sum of all three coordinates is always 0.
