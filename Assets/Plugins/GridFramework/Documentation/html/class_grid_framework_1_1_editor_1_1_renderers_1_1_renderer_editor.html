Detailed Description

Template Parameters
TType of renderer the inspector is for

You should use this class as the base of your own renderer components. It displays the common fields and offers a function to override for your own fields.

You do not have to inherit from this class, but it helps to have uniformity in the look of the inspectors. If you decide to write an inspector from scratch make sure you have fields for the renderer's Meterial, Priority, LineWidth and the axis colours (ColorX, ColorY and ColorZ).
